Hand Gesture Monitoring System (Python + OpenCV)

Description:
-------------
A real-time hand gesture recognition system that detects finger positions via webcam and maps specific gestures to pre-defined actions like opening YouTube or pausing a movie.

Features:
- Used Open CV
- Real-time gesture recognition
- Maps gestures (e.g., pointing index finger) to tasks
- Automates browser or media control actions

Tech Stack:
- Python
- OpenCV
- PyAutoGUI

Status:
Stable basic version; can be extended with more gestures and better control mapping.
